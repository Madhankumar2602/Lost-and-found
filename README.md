# Lost and Found Portal

A web application that helps users report lost items and find recovered belongings efficiently.

## Features
- Report lost items
- Report found items
- Upload images
- Search and filter listings
- Responsive UI
- Real-time backend integration

## Tech Stack
Frontend:
- React
- CSS / Tailwind

Backend:
- Node.js
- Express
- MongoDB

Deployment:
- Vercel
- Render
